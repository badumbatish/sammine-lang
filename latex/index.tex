\doxysubsection*{Dev}

\doxysubsubsection*{Mac\+OS Configurations}

Install llvm \& catch2


\begin{DoxyCode}{0}
\DoxyCodeLine{brew\ install\ llvm\ catch2}

\end{DoxyCode}
 Run 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/S\ .\ -\/B\ build\ -\/DCMAKE\_EXPORT\_COMPILE\_COMMANDS=ON\ -\/DCMAKE\_BUILD\_TYPE=Debug\ [-\/DSAMMINE\_TEST=ON/OFF]\ [-\/DCMAKE\_LINKER\_TYPE=MOLD]}

\end{DoxyCode}


for configuration. We need \href{https://github.com/Shuriken-Group/setup_llvm_tools}{\texttt{ llvm}},\href{https://pypi.org/project/filecheck/}{\texttt{ File\+Check}}, and \href{https://pypi.org/project/lit/}{\texttt{ lit}}.

Run


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/-\/build\ build\ -\/j\ }

\end{DoxyCode}


and


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake\ -\/-\/build\ build\ -\/j\ -\/-\/target\ unit-\/tests\ e2e-\/tests}

\end{DoxyCode}
 in the project root folder to cycle through the development process.

Run 
\begin{DoxyCode}{0}
\DoxyCodeLine{rm\ -\/rf\ build}

\end{DoxyCode}
 to erase the build folder (similar to make clean).
