llvm::AllocaInst * sammine_lang::AST::CgVisitor::CreateEntryBlockAlloca ( llvm::Function * Function,
const std::string & VarName,
llvm::Type * type )

CreateEntryBlockAlloca - Create an alloca instruction in the entry block of the function. This is used for mutable variables etc.

from the llvm blog: mem2reg only looks for alloca instructions in the entry block of the function. Being in the entry block guarantees that the alloca is only executed once, which makes analysis simpler.
